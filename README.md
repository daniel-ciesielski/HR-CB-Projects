## 🇵🇱 Polski


## O projekcie
Zbiór samodzielnie wykonanych analiz z obszaru **Compensation & Benefits** 
dla firmy produkcyjno-handlowej (300 pracowników, 4 obszary biznesowe).

Projekt powstał w ramach rozwoju kompetencji analitycznych w C&B.

## Zawartość
| Arkusz | Opis |
|--------|------|
| 1. Benchmarking | Pozycja płacowa 32 stanowisk vs rynek (P50), compa-ratio, koszt dostosowania |
| 2. Analiza Płac | Statystyki opisowe per obszar — mediana, P25/P75, rotacja, premia |
| 3. Pay Bands | Widełki płacowe per grade i obszar, underpaid/overpaid, range penetration |
| 4. System Premiowy | KPI, cap premii, rekomendacje dla zarządu per obszar |

## Obszary biznesowe
- 🏭 **Produkcja** (71 pracowników)
- 📦 **Magazyn/Logistyka** (58 pracowników)
- 🛒 **Sprzedaż Detaliczna — Retail** (96 pracowników)
- 🏢 **HQ: HR + Finanse + IT + Marketing** (75 pracowników)

## Narzędzia
Excel — tabele przestawne, AVERAGEIF, COUNTIFS, formuły tablicowe, VLOOKUP/XLOOKUP

## ⚠️ Informacja o danych
**Wszystkie dane są fikcyjne** — stworzone wyłącznie do celów demonstracji 
kompetencji analitycznych C&B. Nie pochodzą od żadnego rzeczywistego pracodawcy.

## 🇬🇧 English

### About
A self-made Compensation & Benefits analytics portfolio for a fictional 
production & retail company (300 employees, 4 business areas).  
Built to demonstrate practical C&B skills: benchmarking, pay bands, 
salary analysis and bonus systems.

### Contents
| Sheet | Description |
|-------|-------------|
| 1. Benchmarking | Pay positioning of 32 roles vs market (P50), compa-ratio, adjustment cost |
| 2. Salary Analysis | Descriptive stats per area — median, P25/P75, turnover, bonus |
| 3. Pay Bands | Grade structure per area, underpaid/overpaid analysis, range penetration |
| 4. Bonus System | KPIs, bonus cap, overpaid/underpaid flags, board-level recommendations

### Business Areas
- 🏭 **Production** (71 employees)
- 📦 **Warehouse / Logistics** (58 employees)
- 🛒 **Retail** (96 employees)
- 🏢 **HQ: HR + Finance + IT + Marketing** (75 employees)

### Tools
Excel — pivot tables, AVERAGEIF, COUNTIFS, array formulas, VLOOKUP/XLOOKUP

### ⚠️ Data Disclaimer
**All data in this project is entirely fictional.**  
It was created solely to demonstrate analytical competencies in C&B.  
No data originates from any real employer or employee.
